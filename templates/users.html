How Authentication Works

1. Log in with your credentials via POST /auth/login

2. Receive a JWT token valid for your session.

3. Token is stored locally and sent automatically with all ML requests.

4. Use the Model & Predict page to train and run predictions.

Passwords are stored hashed — never as plain text.

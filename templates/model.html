Model Training & Prediction

First log in with a valid user, then train your model and predict running time based on training hours.

1) Login First

The train and predict endpoints are protected, so you must authenticate first.

Tips for Input

The UI sends JSON to your FastAPI endpoints.

  • Enter numbers separated by commas, for example: 2, 3, 5, 7, 9
  • X and Y must have the same length
  • Degree defaults to 3
  • Prediction uses the logged-in user's saved model file

2) Train Model

This sends a POST /TRAIN request with X, Y and degree.

3) Predict Running Time

This sends a GET /PREDICT/{hours} request using the saved model of the logged-in user.

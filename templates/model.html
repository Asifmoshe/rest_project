Model Training & Prediction

First log in with your user, then train a model using X, Y and degree, and finally predict running time according to training hours.

Users Page Model Page

1) Login

You must log in first. The JWT token is saved in the browser and used for /TRAIN and /PREDICT/{hours}.

Input Tips

Use comma-separated values for training data.

  • X example: 2, 3, 5, 7, 9
  • Y example: 95, 84, 72, 66, 60
  • X and Y must have the same number of values
  • Degree defaults to 3
  • Prediction loads the model of the logged-in user

2) Train Model

Send training data to POST /TRAIN.

3) Predict Running Time

Enter the number of training hours and send a request to GET /PREDICT/{hours}.
